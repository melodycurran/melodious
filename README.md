# Overview

This project is a full-stack music playlist web application built to deepen my software engineering skills in modern JavaScript development. It combines a React + Vite frontend with an Express backend to provide a responsive experience for discovering tracks and managing a temporary playlist.

Melodious lets users browse pop tracks fetched from the Jamendo API, view song details, and add or remove songs from a playlist stored in-memory on the server. The goal is to practice API integration, frontend routing, backend routing, and state management while building a cohesive music discovery experience.

[Software Demo Video](https://youtu.be/qXu9BKjgCN4)

# Development Environment

The software was developed using Node.js and npm, with Visual Studio Code as the primary editor. The frontend uses React, Vite, React Router, and Tailwind CSS. The backend uses Express and CORS to serve API routes and proxy Jamendo requests.
